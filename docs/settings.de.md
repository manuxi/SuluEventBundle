### Event-Einstellungen

Zugriff über Sulu Admin → Events → Einstellungen

### Allgemein

![img.png](img/settings.de.png)

**Einzelansicht:**
- Header, Hero-Bild, Breadcrumbs umschalten
- Tags / Kategorien einblenden

**Listenansicht**
- Verlinkung ermöglichen für
  - Rss-/Atom-Feeds
  - iCal-Download 

### Kalender

![img.png](img/settings-calendar.de.png)

**Grundeinstellungen:**
- Erster Wochentag
- Wochennummern zeigen
- Wochenenden zeigen

**Events-Einstellungen**
- Max. Events pro Tag
- Event-Zeit anzeigen
- Auf Event-Zeitraum begrenzen
- Event-Typ anzeigen
- Event-Ort anzeigen
- Standard Event-Farbe

**Anzeigeoptionen**
- Sicht-Wechsel für Besucher erlauben
- Erlaubte Sichten

**Anzeigeoptionen Wochensicht**
- Startzeit (Standard: 00:00)
- Endzeit (Standard: 23:59)

**Anzeigeoptionen Jahressicht**
- Anzuzeigende Monate (Standard: 3)

### Breadcrumbs

![img.png](img/settings-breadcrumbs.de.png)

**Breadcrumbs:**
- Hier können Seiten definiert werden, die als Zwischenseiten in den Breadcrumbs erscheinen, wenn der entsprechende Filter gesetzt ist.
- Bspw. /veranstaltungen/[Seite mit allen Veranstaltungen]/EinzelVeranstaltung
